## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Miniml Theme"
4. Click Install
5. Press Ctrl+K Ctrl+T (Cmd+K Cmd+T on macOS) to open the theme selector
6. Select "Miniml" from the list
